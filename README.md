This repository contains the official code implementation for our paper: “Bridging Simulation and Reality: A Domain-Agnostic Latent Alignment World Model for End-to-End Autonomous Driving”.
This repository provides the research code associated with our work on latent alignment world models for end-to-end autonomous driving. The project focuses on bridging the gap between simulation and real-world deployment through domain-agnostic latent representation learning.
The code in this repository is currently being organized and cleaned for public release.
Additional components, documentation, and usage instructions will be gradually updated.
